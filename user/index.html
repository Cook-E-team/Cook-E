Cook-E

A mobile application that helps schedule cooking with multiple recipes

Cook-E is a free android application that helps people cook meals easier and faster. It is a perfect choice for people who like cooking and want to cook delicious meal in a shorter time.

Cook-E has many different functionalities that help you save time while enjoying cooking. The most important feature of Cook-E is scheduling, which is the ability to find a step-by-step instruction to cook multiple dishes at the same time. When you select the recipes you want to cook and press the start button, Cook-E will find the best schedule for you to cook those dishes. This helps you combine all steps that use same tools and ingredients, do multiple steps at the same time, and so on. More importantly, Cook-E generate a step-by-step instruction so you only need to follow the steps without worrying about anything.
